Builds

Parts lists and build guides for any quad you'd want!

You'll find parts recomendations and maybe guides too for any kind of build you'd want. They will be sorted into groups depending on the difficulty of the build.
I'm still working on these, as they're finished, they'll be accesible. The ones you can access are finished, the ones you can see are being worked on, and the ones that aren't here are planned
The build lists might be broken on mobile, there's currently not much I can do about it

Easy

- You should be able to build it as a complete beginner. Easy to put together and set up, plenty of space, and parts can be easily found

Medium

- A bit more difficult, but still manageable. Smaller spaces and solder pads, tighter build restrictions, the setup can be harder, and parts are less available

Hard

- For advanced pilots. Either for a very specific use case, or just difficult in general. Long build and setup process, parts pricey, and few in between
